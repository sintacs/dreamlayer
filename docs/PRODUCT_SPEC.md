# Memoscape — Product Spec

## Summary
Memoscape is a private memory layer for Brilliant Labs Halo. It makes the glasses
feel like they *gained memory*: glanceable recall of objects, commitments, and
places, delivered as calm, Apple-like answer cards in-eye.

## North star experience
The user should feel a subtle cognitive extension — trustworthy, elegant,
emotionally impressive, useful in seconds, visually restrained but magical.

## HUD principles
1. One thought at a time
2. Glanceable in under 2 seconds
3. Minimal words
4. High contrast
5. Soft transitions
6. No clutter
7. No dense menus in-eye
8. Premium typography and spacing
9. Strong confidence signaling
10. Always obvious when memory is active vs paused

## Primary use cases
- **Object recall** — "Where did I leave my keys?"
- **Commitment recall** — "What did I promise Jordan?"
- **Proactive place memory** — surfacing prior context when you return somewhere
- **Privacy pause** — instantly, visibly stop capture

## Non-goals (this phase)
- No firmware-level custom OS / Zephyr fork
- No full firmware rewrite
- No raw media hoarding
- No dense dashboards in-eye
- No flashy sci-fi effects
