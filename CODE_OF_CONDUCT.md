# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, in full:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

The one-paragraph version: be kind, be constructive, assume good faith.
Harassment, personal attacks, and discriminatory behavior have no place here,
in any project space — issues, PRs, Discussions, Discord.

Reports: **conduct@dreamlayer.app** (or any private channel to the
maintainer). Reports are handled confidentially. Consequences scale from a
correction to a permanent ban, at the maintainers' judgment, per the
Covenant's enforcement guidelines.

One project-specific addition: the privacy contract (capture guards, the
Veil, no stranger identification) is a values commitment, not just a
technical one. Advocating for weakening it is a design discussion we'll have
respectfully; attempting to sneak it past review is a conduct issue.
